Block Elements

Block elements are elements that start on a new line and take up the full width available.

Examples of block elements include:

  • <div> - A generic block container for flow content.
  • <h1> to <h6> - Define HTML headings.
  • <p> - Defines a paragraph.
  • <ul> - Defines an unordered list.
  • <ol> - Defines an ordered list.
  • <table> - Defines a table.
  • <section> - Defines a section in a document.
  • <article> - Defines an independent, self-contained piece of content.
  • <nav> - Defines a set of navigation links.
  • <footer> - Defines a footer for a document or section.
