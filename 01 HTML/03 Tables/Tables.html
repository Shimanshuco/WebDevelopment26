Tables

Tables are used to display data in a structured format.

<table>
Defines a table.
<tr>
Defines a table row.
<th>
Defines a table header cell.
<td>
Defines a table data cell.
<caption>
Defines a table caption.
<thead>
Defines a table header section.
<tbody>
Defines a table body section.
<tfoot>
Defines a table footer section.
<colgroup>
Defines a group of columns in a table.
<col>
Defines a column in a table.
<rowspan>
Specifies the number of rows a cell should span.
<colspan>
Specifies the number of columns a cell should span.
